# 🎓 Campus Market

A platform for students to buy and sell books, notes, laptops, mobiles and study materials.

## 🚀 Features

* User Registration & Login
* Buy & Sell Products
* Notes Sharing
* Product Search
* Student Marketplace
* Responsive Design

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
