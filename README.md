# 📌 Project Overview

Campus Market is a modern Student Marketplace platform where college students can buy, sell, exchange and manage study materials including:

- 📚 Books
- 📝 Notes
- 💻 Laptops
- 📱 Mobiles
- 🎒 Accessories
- 📂 PDF Notes
- 📄 Documents
- 🎥 Video Courses

The platform provides secure authentication, student verification, admin approval, product management, analytics, chat system, QR code generation and many more advanced marketplace features.
